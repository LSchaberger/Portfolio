# 📊 Data Analyst Portfolio

¡Bienvenido/a a mi portafolio de Data Analytics!

Mi nombre es **Leandro Daniel Schaberger** y este repositorio reúne proyectos desarrollados de principio a fin utilizando datos reales o escenarios basados en problemas reales de negocio.

Cada proyecto está diseñado para simular el trabajo que realiza un Data Analyst en una empresa: comprender el problema, recopilar y preparar los datos, analizarlos, obtener conclusiones y presentar los resultados de forma clara para facilitar la toma de decisiones.

El objetivo de este portafolio no es únicamente demostrar conocimientos técnicos, sino evidenciar mi capacidad para transformar datos en información útil y generar soluciones que aporten valor.

---

# ¿Qué encontrarás en este repositorio?

Los proyectos están organizados por tecnología y área de especialización para facilitar la navegación.

Cada proyecto incluye, cuando corresponde:

* Definición del problema.
* Objetivos del análisis.
* Obtención o descripción de la fuente de datos.
* Limpieza y preparación de los datos.
* Análisis exploratorio.
* Aplicación de técnicas estadísticas o analíticas.
* Desarrollo de dashboards o visualizaciones.
* Conclusiones.
* Recomendaciones basadas en los resultados.

# Tecnologías y herramientas

## SQL

Consultas avanzadas, modelado de bases de datos, optimización de consultas y análisis de información mediante lenguaje SQL.

## Python

Limpieza de datos, automatización de procesos, análisis exploratorio, visualización de datos y desarrollo de soluciones utilizando bibliotecas especializadas.

## R

Análisis estadístico, probabilidad, exploración de datos y aplicación de técnicas cuantitativas.

## Excel

Análisis de datos, tablas dinámicas, funciones avanzadas, gráficos y automatización de tareas.

## Power BI

Creación de dashboards interactivos, indicadores de desempeño (KPIs) y reportes ejecutivos.

## Tableau

Visualización profesional de datos y desarrollo de dashboards orientados al análisis empresarial.

## Looker Studio

Construcción de informes dinámicos conectados a múltiples fuentes de datos.

## Qlik Sense

Desarrollo de dashboards interactivos, análisis asociativo de datos y creación de reportes empresariales para apoyar la toma de decisiones.

## Estadística

Aplicación de métodos estadísticos descriptivos e inferenciales para respaldar decisiones basadas en datos.

## ETL

Procesos de extracción, transformación y carga de datos provenientes de diferentes orígenes.

## Data Cleaning

Preparación y depuración de datos para garantizar información consistente y confiable.

## Data Visualization

Diseño de gráficos y visualizaciones que permitan comunicar información de forma clara y efectiva.

## Web Scraping

Obtención automatizada de información desde sitios web para su posterior análisis.

## Machine Learning

Proyectos introductorios enfocados en modelos predictivos y clasificación.

## Kaggle

Proyectos desarrollados utilizando conjuntos de datos públicos y desafíos de análisis de datos.

# Metodología de trabajo

La mayoría de los proyectos siguen una metodología estructurada que incluye las siguientes etapas:

1. Comprensión del problema.
2. Definición de objetivos.
3. Recolección de datos.
4. Limpieza y transformación de la información.
5. Análisis exploratorio.
6. Aplicación de técnicas estadísticas o analíticas.
7. Desarrollo de visualizaciones y dashboards.
8. Interpretación de resultados.
9. Elaboración de conclusiones.
10. Propuestas de mejora o recomendaciones.
